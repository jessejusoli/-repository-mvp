## Before Asking For Help

Check:

1. [README.md](README.md)
2. [docs/README.md](docs/README.md)
3. [docs/glossary.md](docs/glossary.md)
4. open issues or the active task tracker when available

## Issue Routing

- Use bugs for broken or incorrect behavior.
- Use feature requests for new capabilities or workflow changes.
- Use security reporting for suspected vulnerabilities.
- Use support channels for questions that do not require repository changes.

## Escalation

Escalate when:

- a security issue may affect users or systems;
- a production or operational environment is affected;
- a blocker prevents the current vertical goal;
- ownership is unclear.

Escalation owner: `[ESCALATION_OWNER]`
