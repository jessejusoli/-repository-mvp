# Code Of Conduct

This project expects professional, respectful collaboration from humans, AI agents, and automated systems that participate in the repository workflow.

## Expected Behavior

- Communicate with clarity and respect.
- Assume good intent while still reviewing work carefully.
- Keep feedback specific, actionable, and connected to the work.
- Respect privacy, confidentiality, and security boundaries.
- Help maintain a productive environment for people with different backgrounds and experience levels.

## Unacceptable Behavior

- Harassment, threats, abuse, or discriminatory language.
- Publishing private information without permission.
- Intentionally misleading contributors or reviewers.
- Introducing malicious code, secrets exposure, or unsafe automation.
- Using AI-generated content to impersonate people or bypass review.

## Reporting

Report conduct concerns through `[CONDUCT_REPORTING_CHANNEL]`.

Reports should include:

- what happened;
- when it happened;
- who was involved;
- supporting context when safe to share.

## Enforcement

Project maintainers may remove content, restrict participation, require remediation, or escalate to the project owner when behavior harms the project or its participants.

## Public Project Note

If this repository becomes open source or community-facing, replace or expand this file with a formal community code of conduct selected by the project owner.
